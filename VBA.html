VBA - FUND PRICE AUTOMATION MACRO

This article provides a detailed, step-by-step explanation on my VBA project.
If you would like to review the VBA macro file and the full code script, please visit my GitHub page at here.

In this project, the key objective is to automate the process of updating fund price to enhance efficiency. The data source from this project is fund price data obtained from multiple fund houses. The fund prices are publicly available and can be accessed through the internet.

1. Prepare data sourcce

The fund price data is collected and downloaded into a designated folder. In this project, the fund prices for the date 5/5 are gathered, and it is assumed that a total of 24 fund prices will be updated.

Furthermore, exchange rates will also be updated in this project. The exchange rate data is collected from the HKAB website and the rates are manually inputted into an Excel file named "Fx rate template".

Fx rate template:
2. Input fund price date

Press "Start" button in the VBA macro to execute the macro.

An interactive box will prompt the user to enter the date of the fund price (i.e. 2023-05-05 in this project). A loop mechanism ensures that only valid dates are accepted, prompting the user to re-enter if an invalid date is provided.

3. Merge data source to the excel
Another interactive box will appear, allowing the user to select the desired data source. Once selected, the data source will be merged into the macro excel file as individual spreadsheets.

4. Find fund price from data source

The VBA macro will perform a data search and extraction process across all spreadsheets in the Excel file. It will extract targeted prices based on specified criteria, such as ISIN. If the extraction is successful, the fund prices will be automatically updated.

5. Find exchange rate from data source

The VBA macro will automatically update the exchange rate using the same approach.

6. Save the file as a new excel to a designated path

Upon completion, the file will be automatically saved with a name corresponding to the fund price date, such as "Fund Price Update @2023-05-05".

Remarks: To reuse the macro to update fund price in another date, press "Clear Data" button in the VBA macro to erase all the current data.
